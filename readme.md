# LeadsBridge Landing Page Reproduction

This is a frontend technical assessment project for **SM Technology**. The goal was to replicate a specific Figma design into a pixel-perfect, responsive landing page using modern web technologies.

## 🚀 Live Demo
Live demo: [https://frksarkar.github.io/leadsbridge-ui-clone/](https://frksarkar.github.io/leadsbridge-ui-clone/)

## 🛠️ Technologies Used
- **HTML5**: Semantic structure.
- **Tailwind CSS / CSS3**: For styling and layout.
- **JavaScript**: For interactivity (if any).
- **Figma**: For design reference.

## 📋 Features
- **Pixel Perfect Design**: Closely followed the Figma design (node-id: 1-2).
- **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile screens.
- **Clean Code**: Followed best practices for folder structure and naming conventions.
- **Modern UI**: Implemented modern design elements as per the requirements.
